## 🖥️ What sahin Does

sahin is a Windows-friendly security tool that helps you run network checks from one place. It uses simple workflow files, shows results in a web page, and can make reports in HTML, PDF, and DOCX.

It is built for tasks such as:

- scanning ports on a host
- checking common network paths
- collecting public information
- running Turkey-focused recon checks
- watching for signs of BGP hijack issues
- creating a clear report when the scan ends

## 📝 How to Use It

sahin works with a YAML workflow. A workflow is a small text file that tells the app what to do step by step.

A basic flow can include:

- target discovery
- port checks
- web checks
- OSINT lookups
- report output

For a first run, use a simple workflow and one target only. This helps you see how the app works without extra steps.

Example use:

- enter a host or domain
- run the scan
- watch the progress in the web UI
- open the report when it is done

## ⚙️ Basic Workflow Tips

Use small, clear workflow steps:

- start with one target
- test one module at a time
- save reports after each run
- keep file names short
- use plain target names like `office-router` or `test-domain`

If a workflow fails, check the YAML file first. A missing space or wrong field name can stop the run.

## 🛠️ Common Windows Fixes

If sahin does not open, try these steps:

- right-click the file and choose Run as administrator
- check that the file was fully extracted
- confirm that your antivirus did not block it
- move the app to a short path like `C:\sahin`
- close and reopen your browser if the web UI does not load

If the app starts but the page stays blank, refresh the browser or try another browser.
